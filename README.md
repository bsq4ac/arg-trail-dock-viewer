# arg-trail-dock-viewer
ARG Trail Dock 用 ログビューアー

ブラウザ拡張「ARG Trail Dock」からエクスポートしたバックアップJSON（ARG_Trail_Dock_backup_～.JSON）を読み込み、拡張本体がなくてもプレイ
記録を閲覧できるページです。読み込んだデータはブラウザのローカルストレージにのみ保存され、サーバー等へ保存・送信されることはありません。

「ページ一覧」の「CSVを出力」では、ARG作品ごとに登録したページ情報、遷移ワード、発見キーワードなどをCSV形式で保存できます。

ビューアーはこちら: [viewer.html](https://bsq4ac.github.io/arg-trail-dock-viewer/viewer.html)
